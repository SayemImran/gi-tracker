# Questions Answer
## Answer 01
| Keyword | Description | Scope | Redeclare | Reassign |
|--------|-------------|-------|-----------|----------|
| **var** | Older way of declaring variables in JavaScript | Function-scoped |  Yes |  Yes |
| **let** | Introduced in ES6 for modern variable declaration | Block-scoped |  No |  Yes |
| **const** | Used for variables whose value should not change | Block-scoped |  No |  No |

## Answer 02
The spread operator (...) is used to expand elements of an array or properties of an object. It helps copy or merge data easily.

## Answer 03
| Name | Description |
|--------|-------------|
| **map** |Creates a new array.Each element is transformed based on a function. | 
| **filter** | Creates a new array containing only elements that match a condition |
| **forEach** | Executes a function for each element in an array.It does not return a new array |

## Answer 04
An arrow function is a shorter way to write functions in modern JavaScript. It was introduced in ES6 and uses the => syntax.

```js
//normal function
function add(a, b) { return a + b; }
// arrow function
const add = (a, b) => a + b;
```
